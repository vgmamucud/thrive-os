# The Thrive Vault

The Thrive Vault is the source of truth for Thrive OS: an Obsidian vault holding the constitution, vision, architecture, decisions, specifications, projects, procedures, and AI-collaboration notes that govern the system. Implementation must follow what's documented here, not chat history.

## Structure

- `01 Constitution/` — foundational, rarely-changed principles.
- `02 Vision/` — long-term direction and goals.
- `03 Architecture/` — current, as-built system design.
- `04 ADR/` — Architecture Decision Records.
- `05 Specifications/` — approved specs that implementation must follow.
- `06 Projects/` — active and planned project notes.
- `07 SOP/` — standard operating procedures.
- `08 AI/` — AI collaboration notes and conventions.
- `09 Templates/` — reusable note templates for the folders above.
- `99 Archive/` — retired or superseded content.

See `CLAUDE.md` at the repository root for the full governing instructions, and `CONTRIBUTING.md` for how work flows through this structure.
