# 09 Templates

Reusable note templates for the Vault. Copy the relevant template into its target folder rather than editing these in place.

- `ADR Template.md` — for `04 ADR/`
- `Specification Template.md` — for `05 Specifications/`
- `Project Template.md` — for `06 Projects/`
- `SOP Template.md` — for `07 SOP/`
- `Meeting Notes Template.md` — for meeting records (file under the relevant Project, or `08 AI/` for AI-collaboration syncs)
